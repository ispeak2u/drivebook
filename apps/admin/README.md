# DriveBook Admin App

Admin-facing DriveBook application for marketplace operations.

## Current Scope

- TypeScript starter app placeholder.
- Future home for instructor approvals, review moderation, disputes, and operational dashboards.
- No database, payment, or external API integrations yet.

## Commands

```bash
npm run typecheck --workspace @drivebook/admin
npm run build --workspace @drivebook/admin
```
